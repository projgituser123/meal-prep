json = javascript object notation .. data will be stored in this form.. treat just like dictionery..
syntax :  object_name = {key : value , ...... }
to fetch certain info or the values from the json object use syntax : 
        dot notation : object_name.keys
                     : object_name["keys"]
if writing in json file, keys and values must be in " "
